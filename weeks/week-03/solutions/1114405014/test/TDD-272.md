## 題目概述
UVA 272 是一個文字處理問題，需要將普通雙引號 (") 轉換為 TeX 格式的有方向性雙引號。第一個引號用 `` 代替，第二個用 '' 代替，交替進行。

### TDD 三階段流程
1. **紅燈階段 (Red)**：先寫測試，讓測試失敗
2. **綠燈階段 (Green)**：實現功能，讓測試通過
3. **重構階段 (Refactor)**：優化代碼，保持測試通過

### 測試案例設計原則
- **邊界測試**：測試空字串、無引號等極端情況

## 實現細節

### 核心演算法
```python
def convert_quotes(text):
    result = []
    quote_count = 0

    for char in text:
        if char == '"':
            if quote_count % 2 == 0:
                result.append('``')  # 第一個引號用 ``
            else:
                result.append('\'\'')  # 第二個引號用 ''
            quote_count += 1
        else:
            result.append(char)

    return ''.join(result)
```

### 關鍵設計決策
1. **使用計數器**：`quote_count` 記錄遇到的引號數量
2. **模運算判斷**：偶數索引用左引號，奇數用右引號
3. **字元級處理**：逐個處理字元，確保精確控制
4. **不修改原字串**：返回新字串，保持函數純淨性
