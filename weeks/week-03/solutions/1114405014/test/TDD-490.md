# TDD 測試過程及結果說明 - 題目 490 (UVA 490)

## 題目概述
UVA 490 是一個文字旋轉問題，需要將輸入的文字行順時針旋轉90度。最後一行變成最左列，第一行變成最右列，並保持矩形形狀。

2. **綠燈階段 (Green)**：實現功能，讓測試通過
3. **重構階段 (Refactor)**：優化代碼，保持測試通過

- **邊界測試**：測試空輸入、單行、單字元情況
- **基本功能測試**：測試題目提供的範例

## 實現細節

### 核心演算法
```python
def rotate_text(lines):
    if not lines:
        return []
    
    # 找到最長行的長度，用於填充
    max_length = max(len(line) for line in lines)
    
    # 將所有行填充到相同長度
    padded_lines = [line.ljust(max_length) for line in lines]
    
    # 創建旋轉後的結果
    result = []
    for col in range(max_length):
        # 從最後一行開始，向上收集每一列的字元
        row_chars = []
        for row in range(len(padded_lines) - 1, -1, -1):
            row_chars.append(padded_lines[row][col])
        result.append(''.join(row_chars).rstrip())
    
    return result
```

### 關鍵設計決策
1. **填充策略**：使用 `ljust()` 填充空格到最大長度
2. **旋轉邏輯**：從最後一行開始收集每一列的字元
3. **右對齊移除**：使用 `rstrip()` 移除尾部空格
4. **邊界處理**：正確處理空輸入情況

### 旋轉原理說明
- 原始矩陣的第 i 行第 j 列的字元
- 旋轉後會變成第 j 行第 (行數-1-i) 列的字元
- 通過雙重循環實現這個轉換
