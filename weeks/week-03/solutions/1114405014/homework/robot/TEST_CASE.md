# 測試案例設計說明

本文件列出本專案的主要測試案例，用於驗證 `robot_core.py` 的核心邏輯是否正確。

測試涵蓋以下功能：

- 機器人方向旋轉
- 機器人前進移動
- 地圖邊界判斷
- LOST 狀態處理
- scent（氣味）機制
- 指令序列執行
- 非法指令錯誤處理

測試程式使用 Python `unittest` 框架實作。

# 測試覆蓋範圍

本測試案例覆蓋以下核心邏輯：

### 方向控制
- 左轉
- 右轉
- 多次旋轉

### 移動邏輯
- 正常前進
- 多指令序列執行

### 邊界判斷
- 地圖邊界偵測
- 掉落（LOST）判斷

### scent 機制
- scent 建立
- scent 避免重複掉落
- scent 與方向相關
- scent 不影響其他移動

### 錯誤處理
- 非法單一指令
- 指令序列中的非法指令

# 測試執行方式

執行以下指令即可執行所有測試：

```bash
python -m unittest discover -s tests -p "test_*.py" -v
```

測試結果：

```
Ran 18 tests in 0.002s

OK
```
