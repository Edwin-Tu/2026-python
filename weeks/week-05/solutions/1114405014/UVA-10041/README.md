# UVA-10041 - Vito's Family

## 📌 題目說明

Vito 想要在一條直線街道上買房子，使得所有親戚到他家的距離總和最小。

給定每位親戚的門牌號碼（整數），請找出最佳位置，使距離總和最小，並輸出該最小距離。

------------------------------------------------------------------------

## 🧠 核心觀念

👉 最小化絕對距離總和 → 最佳解為「中位數（Median）」

------------------------------------------------------------------------

### 2️⃣ Green

實作最小可通過版本：

``` python
def min_total_distance(addresses):
    sorted_addresses = sorted(addresses)
    median = sorted_addresses[len(sorted_addresses) // 2]
    return sum(abs(a - median) for a in sorted_addresses)

def solve(data):
    tokens = data.split()
    t = int(tokens[0])
    idx = 1
    results = []

    for _ in range(t):
        r = int(tokens[idx])
        idx += 1
        addresses = list(map(int, tokens[idx:idx + r]))
        idx += r
        results.append(str(min_total_distance(addresses)))

    return "\n".join(results) + "\n"
```
