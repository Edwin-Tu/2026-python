# UVA-10056 - What is the Probability ?

## 📌 題目說明

有 N 位玩家依序嘗試某個事件，每次成功機率為 `p`。\
一旦有玩家成功，遊戲立即結束。

問題：求第 `i` 位玩家最終獲勝的機率（輸出至小數點後四位）。

------------------------------------------------------------------------

## 🧠 核心觀念

### 🎯 關鍵觀察

第 i 位玩家「第一次成功」的機率：

    (1 - p)^(i-1) * p

代表前 i-1 位都失敗，第 i 位成功。

------------------------------------------------------------------------

### 🔁 無限輪迴（等比級數）

如果第一輪沒人成功，會進入第二輪、第三輪...

每一輪失敗機率：

    (1 - p)^n

因此總機率為：

    first_round + first_round * cycle_fail + first_round * (cycle_fail)^2 + ...

👉 等比級數公式：

    P = first_round / (1 - cycle_fail)

### 🟢 Green

實作最小可通過版本：

``` python
def win_probability(n: int, p: float, i: int) -> float:
    if p == 0.0:
        return 0.0

    first_round = ((1 - p) ** (i - 1)) * p
    cycle_fail = (1 - p) ** n

    return first_round / (1 - cycle_fail)
```

``` python
def solve(data: str) -> str:
    tokens = data.split()
    t = int(tokens[0])
    idx = 1
    results = []

    for _ in range(t):
        n = int(tokens[idx]); idx += 1
        p = float(tokens[idx]); idx += 1
        i = int(tokens[idx]); idx += 1

        results.append(f"{win_probability(n, p, i):.4f}")

    return "\n".join(results) + "\n"
```
