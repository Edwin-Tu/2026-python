# UVA-10050 - Hartals

## 📌 題目說明

本題模擬政黨罷會（hartal）對工作天的影響。

給定： - 模擬天數 N（從星期天開始） - 多個政黨，每個有一個 hartal 參數
h（每 h 天罷會一次）

規則： - 星期五與星期六為假日，不計入罷會損失 -
同一天若多個政黨罷會，只計算一次

👉 目標：計算 N 天內損失的「工作天數」

------------------------------------------------------------------------

## 🧠 核心觀念

1.  每個政黨會在：

        h, 2h, 3h, ...

    這些日子罷會

2.  需要排除：

    -   星期五（day % 7 == 6）
    -   星期六（day % 7 == 0）

3.  使用 `set`：

    -   避免重複計算同一天

### 🟢 Green

實作最小可通過版本：

``` python
def is_weekend(day: int) -> bool:
    return day % 7 == 6 or day % 7 == 0


def count_hartals(n: int, parties: list[int]) -> int:
    hartal_days = set()

    for h in parties:
        for day in range(h, n + 1, h):
            if not is_weekend(day):
                hartal_days.add(day)

    return len(hartal_days)


def solve(data: str) -> str:
    tokens = data.split()
    t = int(tokens[0])
    index = 1
    results = []

    for _ in range(t):
        n = int(tokens[index])
        index += 1

        p = int(tokens[index])
        index += 1

        parties = []
        for _ in range(p):
            parties.append(int(tokens[index]))
            index += 1

        results.append(str(count_hartals(n, parties)))

    return "\n".join(results) + "\n"
```

## 🧩 解題步驟

1.  讀取測資數量 T
2.  每組讀取：
    -   天數 N
    -   政黨數 P
    -   每個 hartal 參數
3.  模擬每個政黨罷會日
4.  排除星期五與星期六
5.  用 set 去重
6.  回傳總數
