# UVA-10055 - Function Monotonicity

## 📌 題目說明

有 n 個函數，初始皆為「遞增函數」。

操作有兩種：

-   `1 i`：將第 i 個函數反轉（遞增 ↔ 遞減）
-   `2 L R`：查詢從 L 到 R 的函數複合後是遞增還是遞減
    -   遞增輸出 `0`
    -   遞減輸出 `1`

------------------------------------------------------------------------

## 🧠 核心觀念

將函數性質轉換為：

-   遞增 → `0`
-   遞減 → `1`

### 關鍵觀察

👉 複合函數的增減性取決於「遞減函數的數量奇偶」

-   偶數個遞減 → 最終為遞增（0）
-   奇數個遞減 → 最終為遞減（1）

### 🟢 Green

實作最小可通過版本：

``` python
def flip_function(states, index):
    states[index - 1] ^= 1


def query_monotonicity(states, left, right):
    return sum(states[left - 1:right]) % 2


def solve(data):
    tokens = data.split()
    n, q = int(tokens[0]), int(tokens[1])
    idx = 2

    states = [0] * n
    results = []

    for _ in range(q):
        op = int(tokens[idx])
        idx += 1

        if op == 1:
            i = int(tokens[idx])
            idx += 1
            flip_function(states, i)
        else:
            l = int(tokens[idx])
            r = int(tokens[idx + 1])
            idx += 2
            results.append(str(query_monotonicity(states, l, r)))

    return "\n".join(results) + "\n"
```

## 🧩 解題步驟

1.  初始化所有函數為遞增（0）
2.  遇到操作：
    -   `1 i` → 翻轉第 i 個函數
    -   `2 L R` → 計算區間內 `1` 的數量
3.  判斷奇偶輸出結果

## 💡 範例
