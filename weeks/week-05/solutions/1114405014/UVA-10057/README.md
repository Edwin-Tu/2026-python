# UVA-10057 - A Mid-Summer Night's Dream

## 📌 題目說明

給定一組整數資料 `X1, X2, ..., Xn`，需要找到一個整數 `A`，使得：

    |X1 - A| + |X2 - A| + ... + |Xn - A|

的值最小。

並輸出三個數值：

1.  最小的最佳整數 `A`
2.  有多少個輸入值落在最佳解範圍內
3.  有多少個不同的整數 `A` 可以達到最小值

------------------------------------------------------------------------

## 🧠 核心觀念

這題本質是 **絕對值最小化問題（Median）**

### 🎯 關鍵結論

-   若 `n` 為奇數 → 最佳解只有一個（中位數）
-   若 `n` 為偶數 → 最佳解為一個區間 `[left_median, right_median]`

def analyze_password(nums):
    nums.sort()
    n = len(nums)

    left = nums[(n - 1) // 2]
    right = nums[n // 2]

    best_a = left
    count = sum(1 for x in nums if left <= x <= right)
    ways = right - left + 1

    return best_a, count, ways

## 🧩 解題步驟

1.  將輸入排序
2.  找出：
    -   左中位數
    -   右中位數
3.  計算：
    -   最小 A（取左中位數）
    -   落在區間的數量
    -   區間長度（解的個數）
