## 10189 - Minesweeper

### 題目資訊
- **題名**: UVA 10189 — Minesweeper
- **相關連結**: 
  - [ZeroJudge 題目頁面](https://zerojudge.tw/ShowProblem?problemid=a182)
  - [Yui Huang 題解](https://yuihuang.com/zj-a182/)

### 題目敘述
在 n 行 m 列的網格中，某些格子放有地雷（以 `*` 表示），其餘格子為空白（以 `.` 表示）。對於每個空白格子，需要計算其周圍 8 個方向中地雷的數量，並以該數字填入。

### 輸入說明
- 輸入包含多組測試資料。
- 每組測試資料第一行為兩個整數 n 和 m（表示網格的行數和列數）。
- 接下來 n 行，每行 m 個字元（`*` 表示地雷，`.` 表示空白）。
- 以 n = 0, m = 0 結束輸入。

### 輸出說明
- 對每組測試資料，先輸出 `Field #X:`（X 為組號，從 1 開始）。
- 地雷格子保持 `*`，空白格子改為周圍 8 格中地雷的數量。
- 每組測試資料之間輸出一個空行。

### 解題思路
1. 讀取測試資料，使用計數器追蹤當前為第幾組測試資料
2. 對每個格子進行判斷：
   - 若為地雷 (`*`)，直接輸出 `*`
   - 若為空白 (`.`)，計算周圍 8 個方向的地雷數量
3. 使用 8 方向偏移陣列，檢查每個相鄰格是否為地雷
4. 格式化輸出：每組測試資料前輸出 `Field #X:`，組間輸出空行

### 測試用例
```
輸入:
4 4
*...
....
.*..
....
3 5
**...
.....
.*...
0 0

輸出:
Field #1:
*100
2210
1*10
1110

Field #2:
**100
33200
1*100
```

## 10190 - 自動傘

### 題目資訊
- **題名**: UVA 10190 - 自動傘
- **相關連結**: [ZeroJudge 題目頁面](https://zerojudge.tw/ShowProblem?problemid=a183)

### 題目敘述
多把自動傘在馬路上方來回移動擋雨。每把傘有長度 l 和速度 v，寬度等於馬路寬度 W。計算從開始下雨到 T 秒後，落到人行橫道上的雨水總體積。

### 輸入說明
- 第一行: N, W, T, V
  - N: 自動傘數目
  - W: 馬路寬度
  - T: 統計時間
  - V: 單位面積單位時間降雨體積
- 接下來 N 行: x, l, v (初始位置, 長度, 速度)

### 輸出說明
輸出雨水總體積，精確到小數點後第二位。

### 解題思路
1. 計算每把傘在任意時刻的位置（來回往返運動）
2. 模擬每個時間點，合併重疊的傘覆蓋區間
3. 計算每個時間點的平均覆蓋長度
4. 計算未覆蓋長度，乘以時間和降雨強度得到體積

## 10193 - Arctan 分解

### 題目資訊
- **題名**: UVA 10193 - All That Loves Chocolate?
- **相關連結**: [ZeroJudge 題目頁面](https://zerojudge.tw/ShowProblem?problemid=a186)

### 題目敘述
找尋整數 b, c 使得 1/a = 1/b + 1/c，其中 a, b, c 均為正整數。求 b + c 的最小值。

### 輸入說明
一個正整數 a (1 ≤ a ≤ 60000)

### 輸出說明
輸出 b + c 的最小值

### 解題思路
由 1/a = 1/b + 1/c 可推導：
- 令 k = b - a，則 c = a + a²/k
- b + c = a + k + a + a²/k = 2a + k + a²/k
- 當 k = a 時，得到最小值 b + c = 4a

因此最小解為 b = 2a, c = 2a，b + c = 4a。

### 測試用例
```
輸入: 1 2 3 4 5 10
輸出: 4 8 12 16 20 40
```

## 10221 - Satellites

### 題目資訊
- **題名**: UVA 10221 — Satellites
- **相關連結**: [ZeroJudge 題目頁面](https://zerojudge.tw/ShowProblem?problemid=a214)

### 題目敘述
地球半徑為 6440 公里。兩顆衛星與地球圓心形成角度 a，計算弧長和弦長距離。

### 輸入說明
多行，每行包含 s（衛星距地表高度）、a（夾角）、unit（deg 或 min）

### 輸出說明
每行輸出弧長和弦長，保留小數點後六位。

### 解題思路
- 半徑 r = 6440 + s
- 若 unit 為 deg，角度轉弧度：a_rad = a * π / 180
- 若 unit 為 min，角度轉弧度：a_rad = (a/60) * π / 180
- 弧長 = r × a_rad
- 弦長 = 2 × r × sin(a_rad / 2)

### 測試用例
```
輸入:
500 30 deg
700 60 min
200 45 deg

輸出:
3633.775503 3592.408346
124.616509 124.614927
5215.043805 5082.035982
```

## 10222 - Decode the Mad man

### 題目資訊
- **題名**: UVA 10222 — Decode the Mad man
- **相關連結**: [ZeroJudge 題目頁面](https://zerojudge.tw/ShowProblem?problemid=a215)

### 題目敘述
瘋狂的人打字時手向右偏移了若干按鍵。將加密文字解碼還原成原本應輸入的文字。

### 鍵盤佈局
```
第一排：` 1 2 3 4 5 6 7 8 9 0 - =
第二排：q w e r t y u i o p [ ] \
第三排：a s d f g h j k l ; '
第四排：z x c v b n m , . /
```

### 輸入說明
一行，包含學生的編號 id（2 ≤ id ≤ 10000）

### 輸出說明
若該名學生為優質學生，輸出 `yes`，否則輸出 `no`

### 解題思路
1. 根據 QWERTY 鍵盤配置，建立每行的字元映射
2. 對於每個輸入字元，找到其所在行與位置
3. 向左偏移一位還原原文
4. 若字元不在鍵盤行中，則保持不變
