## 10193 - Arctan 分解

### 數學推導
由 1/a = 1/b + 1/c 推導：
- 1/b + 1/c = (c + b) / (bc) = 1/a
- ac + ab = bc → bc - ab - ac = 0 → (b - a)(c - a) = a²
- 令 k = b - a，則 c - a = a²/k，c = a + a²/k
- b + c = a + k + a + a²/k = 2a + k + a²/k
- 當 k = a 時，b + c = 2a + a + a = 4a 為最小值

### 驗證數學公式
